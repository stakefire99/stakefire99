## 📌 Project Description
This project involves analyzing restaurant data to extract meaningful insights related to ratings, cuisines, geographic distribution, pricing, and customer engagement.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Folium

## 📊 Levels Completed
- Level 2: Data Analysis
- Level 3: Advanced Analysis

## 🔍 Key Insights
- Most restaurants fall in the 3–4 rating range
- Niche cuisines tend to have higher ratings
- Restaurants cluster in urban areas
- Higher ratings generally receive more votes
- Higher price ranges tend to have better ratings

## 📁 Files Included
- `Level2_Level3.ipynb` → Main analysis notebook  
- `Dataset.csv` → Dataset used  
- `visuals/` → Graphs and plots  

## 📌 Conclusion
The project highlights how multiple factors such as pricing, location, and cuisine influence restaurant performance and customer perception.
